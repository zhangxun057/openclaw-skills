# 客户类型分类指南

## 6种客户类型定义

### 1. 体系伙伴
- 定义：与酒交所有深度战略合作关系的伙伴
- 特征：长期合作、资源互补、战略协同
- 示例：战略投资方、核心供应商

### 2. 市场服务商
- 定义：为酒交所提供市场服务的机构
- 特征：专业服务能力、市场资源
- 示例：营销公司、渠道商、代理商

### 3. 创始会员
- 定义：酒交所创立时的初始会员
- 特征：早期参与、特殊权益
- 识别：明确的创始会员身份

### 4. 其他会员
- 定义：普通会员、合伙会员、交易会员等非创始会员
- 特征：已注册会员身份，但非创始会员
- 包含：所有非创始的会员类型

### 5. 定制酒客户
- 定义：有定制酒需求但不是会员的客户
- 特征：纯业务关系，未加入会员体系
- 识别：有定制需求 + 非会员

### 6. 潜在客户
- 定义：尚未成为会员或客户，但有合作意向
- 特征：初次接触、意向阶段
- 识别：首次拜访、尚未签约

## 分类判断流程

1. 是否已是会员？
   - 是 → 是否创始会员？
     - 是 → **创始会员**
     - 否 → **其他会员**
   - 否 → 继续判断

2. 是否有明确合作关系？
   - 战略合作 → **体系伙伴**
   - 市场服务 → **市场服务商**
   - 定制酒业务 → **定制酒客户**
   - 无 → **潜在客户**

## 访谈中的确认话术

- "请问贵公司是酒交所的会员吗？"
- "是创始会员还是其他类型会员？"
- "您和酒交所目前是什么合作关系？"
