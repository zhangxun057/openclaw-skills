# 客户背景信息搜索指南

## 搜索目标

关注消费实力和销售机会相关信息：

### 1. 企业规模指标
- 上市公司（A股/港股/美股）
- 注册资本
- 员工人数/社保缴纳人数
- 年营收规模

### 2. 行业地位
- 行业排名
- 市场份额
- 知名度/品牌影响力
- 获奖荣誉

### 3. 财务实力
- 纳税大户（地方/省级/国家级）
- 纳税金额
- 利润水平
- 融资情况

### 4. 业务特征
- 主营业务
- 客户群体
- 销售渠道
- 市场覆盖范围

## 搜索策略

### 第一步：基础搜索
```
"{客户名称} 公司 上市"
"{客户名称} 纳税大户"
"{客户名称} 员工人数"
```

### 第二步：深度搜索
```
"{客户名称} 年营收"
"{客户名称} 行业地位"
"{客户名称} 企业规模"
```

### 第三步：确认信息
如搜索结果有多个同名公司或信息不确定，在访谈中向用户确认：
- "搜索到有几家叫{客户名称}的公司，请问是{地区}的那家吗？"
- "了解到{客户名称}是{行业}的企业，对吗？"

## 信息提取重点

从搜索结果中提取：
1. **最关键的1-2个标签**（如"A股上市公司"、"省级纳税大户"）
2. **具体数据**（如"年营收50亿"、"员工3000人"）
3. **行业地位**（如"行业前三"、"区域龙头"）

## 输出格式

简洁、关键信息优先：
- ✅ "A股上市公司，年营收50亿，员工3000人"
- ✅ "省级纳税大户，区域白酒龙头企业"
- ❌ "该公司成立于2005年，主要从事..."（过于冗长）

## 特殊情况处理

- **搜索无结果**：记录"未查到公开信息"，在访谈中询问
- **信息过时**：优先使用最新年份数据
- **多个来源冲突**：选择官方/权威来源
