# WeChat Analyzer - 微信客户自动化分析工具

自动化分析微信好友，生成深度客户档案，助力私域运营变现。

## 快速开始

### 一键安装并运行

```bash
python ~/.openclaw/workspace/skills/wechat-analyzer/scripts/full_pipeline.py \
  --output ~/.openclaw/workspace/客户档案 \
  --download-media \
  --download-moments
```

此命令会自动：
1. 检查并下载WeFlow（如未安装）
2. 启动WeFlow API服务
3. 等待微信数据库连接
4. 下载全量微信数据（含媒体文件、朋友圈）
5. 逐一分析每位客户
6. 生成标准化客户档案

## 功能特性

- **自动集成WeFlow**：自动下载、安装、启动WeFlow服务
- **全量数据下载**：联系人、聊天记录、群聊、媒体文件、朋友圈
- **客户逐一盘点**：为每位客户生成详细档案
- **智能客户分级**：核心/活跃/普通/待激活四级分类
- **消费特征推断**：基于聊天内容分析消费偏好
- **触达策略建议**：针对不同客户类型的沟通建议

## 档案输出

```
客户档案/
├── 核心客户/          # 高频互动，强关系
├── 活跃客户/          # 中频互动，有潜力
├── 普通客户/          # 低频互动，需维护
├── 待激活客户/        # 长期沉默，待激活
└── 客户盘点统计.json   # 统计报告
```

每个客户档案包含：
- 基础画像（关系类型、互动数据）
- 社交强度分析（消息比例、互动频率）
- 互动模式分析（性格、情感倾向）
- 消费特征推断（消费层级、偏好）
- 价值评估（直接/间接价值、网络位置）
- 触达策略建议（时机、方式、注意事项）
- 行动建议（立即/持续/关键节点）

## 分步执行

如需分步执行，可使用以下脚本：

```bash
# 1. 启动WeFlow服务
python scripts/weflow_manager.py --action start --wait-wechat

# 2. 下载全量数据
python scripts/download_all.py \
  --output ./微信数据 \
  --include-media \
  --include-moments

# 3. 客户逐一盘点
python scripts/analyze_all.py \
  --data-dir ./微信数据 \
  --output ./客户档案
```

## 客户分级标准

| 级别 | 标准 | 特征 |
|------|------|------|
| **核心客户** | 消息量>500，互动高频 | 深度关系，强信任 |
| **活跃客户** | 消息量200-500，互动中频 | 活跃联系，有潜力 |
| **普通客户** | 消息量50-200，互动低频 | 一般联系 |
| **待激活客户** | 消息量<50，长期沉默 | 弱关系 |

## 客户类型分析

- **商务型**：关注项目、合作、投资，决策力强
- **技术型**：关注产品、开发、技术，理性分析
- **生活型**：关注家庭、日常，重视体验
- **社交型**：活跃、热情，重视人际关系
- **情感型**：表达丰富，重视情感连接

## 隐私与安全

- 所有数据本地存储，不上传云端
- 客户档案建议加密存储
- 敏感信息（手机号、地址）自动脱敏
- 建议定期清理原始聊天记录
