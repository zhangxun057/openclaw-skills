# Project Manager Skill

规范化项目开发流程，自动创建项目结构、生成文档模板、跟踪开发进度。

## 功能

- ✅ 初始化项目（标准目录结构）
- ✅ 文档模板（PRD、迭代计划、测试报告、ADR）
- 🚧 创建迭代（脚本待完善）
- 🚧 记录决策（脚本待完善）

## 使用方法

### 初始化新项目
```bash
cd skills/project-manager
node scripts/init-project.js my-project
```

### 手动创建文档
使用 `templates/` 目录下的模板文件

## 项目结构规范

详见 `templates/PROJECT-STRUCTURE.md`
