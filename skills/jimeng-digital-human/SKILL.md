# 即梦数字人 Skill

基于火山引擎即梦 OmniHuman1.5 API，实现图片+音频→数字人口播视频。

## 工作流

1. 活体识别 — 检测图片是否包含人像（~10s）
2. 视频生成 — 提交生成任务，60s 间隔轮询（通常 6-12 分钟）
3. 下载持久化 — 视频下载到本地，上传文件服务器获取永久链接
4. 通知 — 输出结构化结果，由 agent 发送通知
