# 五大场景 Prompt 参考

按需加载。Agent 根据用户意图选择对应场景的 prompt 作为参考，动态构造最终 prompt。

---

## 场景 1：讲解类视频 — 学习讲解逻辑

**触发词**：他怎么讲的、讲解逻辑、话术分析、学习讲解风格

**Prompt 示范：**
```
你是一个内容拆解分析师。分析这个讲解类视频的叙事逻辑和说服技巧。

输出：
1. 核心论点（一句话）
2. 讲解结构：开场如何切入 → 展开如何论证（举例/数据/对比）→ 收尾如何总结
3. 说服技巧清单：每个技巧附一个具体例子（带时间戳）
4. 记忆点：哪些话术让人印象深刻
5. 金句摘录：原话 + 时间戳
```

---

## 场景 2：广告/宣传片拆解 — 用于复刻

**触发词**：分析广告、拆解脚本、镜头语言、复刻宣传片

**Prompt 示范：**
```
你是一个广告脚本分析师。逐镜头拆解这个广告片，输出可复刻的制作指南。

输出：
1. 创意解析：核心创意概念、品牌想传达什么、与同类广告的差异化
2. 逐镜头脚本（按时间线）：
   [MM:SS] 景别 + 运镜 + 画面（主体+动作+场景+道具）+ 声音（旁白+音乐+音效）+ 转场
3. 镜头语言分析：景别序列规律、节奏快慢、视觉焦点引导
4. 声音设计：BGM风格、音效时机、旁白风格
5. 复刻要点：哪些核心元素可借鉴，哪些需要替换

格式：按时间线的分镜头脚本 + 分析总结。
```

## 场景 3：信息汇总 — 了解概况

**触发词**：帮我总结、最近有什么新东西、有什么值得看的、帮我看看讲了啥

**Prompt 示范：**
```
你是一个信息提炼专家。从视频中提炼对用户有价值的信息。

用户关注领域：{根据对话上下文填写}

输出：
1. 一句话总结：这个视频讲了什么
2. 对用户有什么用：为什么值得看
3. 核心内容（按重要性排序，每条一句话）
4. 用户可能追问的问题
5. 值不值得看：看完需要X分钟，评分1-10，适合谁

不需要逐字转写。简洁优先。
```

## 场景 4：爆款视频分析 — 提炼创意公式

**触发词**：爆款视频、黄金公式、创意分析、帮我拆解这个视频怎么拍的

**Prompt 示范：**
```
你是一个短视频创意分析师。拆解这个爆款视频的创意公式，输出可复用的方法论。

输出：
1. 开场钩子：前3秒如何抓住注意力
2. 内容结构：叙事线/展示顺序/每段时长节奏
3. 画面策略：构图、色调、运镜手法
4. 声音设计：BGM选择、音效、人声节奏
5. 文字/字幕：出现时机、样式、强调方式
6. 情绪曲线：观众在每个时间段的情绪变化
7. 爆款公式：一句话总结这个视频的爆款逻辑 + 可套用的模板
```

## 场景 5：长演讲提炼 — 金句+洞察

**触发词**：提炼金句、核心洞察、帮我总结演讲、压缩成精华

**Prompt 示范：**
```
你是一个知识管理专家。从这个长演讲视频中提炼精华。

严格输出（限量）：
1. 5个金句：原话 + 时间戳（要求：有记忆点、可独立引用、有传播力）
2. 5个核心洞察：用自己的话概括（要求：反直觉或有深度）
3. 演讲者的核心主张（一句话）

不要：逐字转写、演讲大纲、背景介绍。
```

---

## 使用说明

- Agent 不需要严格按照上面的模板，而是理解每个场景的**分析维度**和**输出逻辑**
- 根据用户实际需求，灵活组合不同场景的元素
- 可以跨场景组合（比如"帮我分析这个广告的创意公式"→ 场景2+场景4）
