# OpenClaw Skills Repository

张洵的多龙虾技能共享库

---

## 技能列表

| 技能 | 用途 | 文件 |
|------|------|------|
| **Gateway 重启** | 配置修改后重启服务 | `skill-gateway-restart.md` |
| **浏览器自动化** | 自动抓取网页、截图、点击 | `skill-browser-automation.md` |
| **Cloudflare 部署** | GitHub → Cloudflare Pages 自动部署 | `skill-cloudflare-pages-deploy.md` |

---

## 使用方式

1. 复制技能文件到你的工作区 `~/.openclaw/workspace/`
2. 按技能文档操作
3. 创建你自己的技能，命名规范 `skill-{功能}.md`
4. 提交 PR 共享给团队

---

## 命名规范

```
skill-{功能}-{细分}.md

示例：
skill-browser-automation.md
skill-cloudflare-pages-deploy.md
skill-gateway-restart.md
```
