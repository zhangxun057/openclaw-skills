# SKILL.md - 定时任务创建技能

## 触发条件
用户要求创建定时任务、设置提醒、预约执行任务时使用此skill。

## 功能
解析用户指令，自动创建多个定时任务（cron job）。

## 使用方式
当用户说"创建X个定时任务"或"在X日期X时间执行XX任务"时：
1. 解析用户给出的所有时间点和任务内容
2. 为每个时间点创建一个独立的cron job
3. 使用cron add创建任务

## 参数提取
从用户指令中提取：
- **执行时间**：具体日期和时间（如3月18日9点、2026年3月21日09:00）
- **任务内容**：要执行的操作（如"生成视频"、"发送消息"）
- **可选：任务名称**：用于标识任务

## cron job格式
```json
{
  "name": "任务名称",
  "schedule": {
    "kind": "at",
    "at": "2026-03-18T09:00:00+08:00"
  },
  "payload": {
    "kind": "agentTurn",
    "message": "执行任务的具体指令",
    "model": "minimax-portal/MiniMax-M2.5",
    "thinking": "off"
  },
  "sessionTarget": "isolated",
  "enabled": true
}
```

## 输出
创建完成后，返回创建的任务列表，包含：
- 任务名称
- 执行时间
- 任务状态

## 示例
用户输入："创建三个定时任务，在3月18日、3月21日和3月24日的9点分别执行视频生成任务"

解析后创建3个cron job：
1. 视频生成任务-2026-03-18 09:00
2. 视频生成任务-2026-03-21 09:00  
3. 视频生成任务-2026-03-24 09:00
