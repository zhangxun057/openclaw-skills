结论

如何选择？

• CLI：适合高频工具，本地安装一次长期用
• MCP：适合 AI Agent 场景，协议标准化，Server 可部署在任意环境
• API：最通用，适合所有编程语言和开发场景

飞书/高德的 CLI 本质是封装了 API 调用的本地二进制，不是必须品，但装了更方便。
