# 战略思维分析

## 毛泽东战略思维的核心特征

### 1. 从弱势出发的战略家

毛泽东思维框架最独特之处：几乎所有分析都是**从弱势方的视角出发**。

- 革命初期：几百人的队伍 vs 几十万人的围剿
- 抗日战争：农业国 vs 工业强国
- 解放战争：土枪 vs 美式装备

这使得他的框架天然适用于：创业公司 vs 大厂、新人 vs 老手、小团队 vs 大组织。

### 2. 辩证思维的极致运用

毛泽东不做线性分析。他的分析永远包含：
- 事物的两面性（一分为二）
- 矛盾的转化条件（什么条件下弱变强、强变弱）
- 阶段性变化（同一事物在不同阶段性质不同）

### 3. 时间维度的战略家

西方商业战略家倾向于"空间思维"——市场定位、竞争格局。
毛泽东更擅长"时间思维"——事物在时间中如何发展、转化、演变。
《论持久战》是这种"时间思维"的典范。

## 战略框架应用对照表

| 毛泽东原始语境 | 现代应用场景 | 核心逻辑 |
|--------------|------------|---------|
| 井冈山根据地 | 创业MVP/种子用户 | 先在小范围验证模式 |
| 农村包围城市 | 下沉市场→一线市场 | 避开强敌主场，从边缘突破 |
| 抗日统一战线 | 竞合关系/行业联盟 | 在共同利益上团结，保持独立性 |
| 论持久战三阶段 | 产品生命周期 | 防御→相持→反攻的节奏把控 |
| 集中优势兵力 | 资源聚焦 | 不在多条战线同时作战 |
| 人民内部矛盾 | 团队冲突管理 | 区分"要消灭的"和"要争取的" |
| 批评与自我批评 | 复盘文化 | 不追责，找原因，改进 |
| 调查研究 | 用户调研/市场调研 | 决策前必须有一手数据 |

## 典型决策模式

### 模式一：矛盾判断 → 策略制定

1. 列出当前所有矛盾（问题）
2. 判断哪个是主要矛盾
3. 分析主要矛盾的两个方面
4. 找到改变矛盾力量对比的条件
5. 制定策略作用于这些条件

### 模式二：敌我友分析 → 联盟构建

1. "谁是我们的敌人？谁是我们的朋友？"
2. 按照与主要矛盾的关系分类
3. 团结朋友，孤立敌人
4. 在联盟内部保持独立性

### 模式三：调查 → 判断 → 试点 → 推广

1. 深入调查了解实际情况
2. 基于调查做出判断
3. 在小范围试点验证
4. 总结经验后逐步推广
5. 在推广中继续调查和修正
