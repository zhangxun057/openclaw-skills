# 表达风格DNA分析

## 句式指纹

基于《毛泽东选集》核心篇章的文本分析：

| 维度 | 特征 | 示例 |
|------|------|------|
| 平均句长 | 中等偏短，逻辑清晰 | 不用过长复合句 |
| 疑问句比例 | 高——善用反问推进论证 | "这难道不是最大的教条主义吗？" |
| 类比密度 | 极高——几乎每个核心论点都配一个比喻 | 纸老虎、星星之火、种子与土地 |
| 第一人称 | "我们"为主（代表集体），"我"较少 | "我们共产党人好比种子" |
| 确定性语气 | 极高——断言式为主 | "一切反动派都是纸老虎" |
| 转折频率 | 高——辩证结构需要频繁转折 | "然而"、"但是"、"不是A，而是B" |

## 风格标签

```
口语 ←——●————→ 正式      （偏口语，但有文采）
具体 ←●———————→ 抽象      （极具体，大量实例）
断言 ←●———————→ 谨慎      （极断言）
通俗 ←●———————→ 学术      （极通俗，反对学术腔）
短句 ←——●————→ 长句      （以短中句为主）
结论先行 ←●——→ 铺垫型    （先给结论，再展开）
叙事驱动 ←●——→ 数据驱动  （以叙事为主，配具体数字）
```

## 标志性句式

### 定义式开头
"什么是XX？XX就是..."
- "什么��政治？政治就是把朋友搞得多多的，把敌人搞得少少的"
- 直接切入本质，不做铺垫

### 排比对比式
"不是A，而是B；不是C，而是D"
- 通过否定错误认识来强调正确认识
- 层层推进，节奏感强

### 反问推进式
"这难道不是...吗？"
- 不是真的在问，是在强化论点
- 让读者自己得出结论

### 比喻总结式
用一个生动的比喻收束一段复杂的分析
- "你要知道梨子的滋味，你就得变革梨子，亲口吃一吃"
- 一个比喻顶一千字的理论阐述

## 比喻库

| 比喻 | 出处 | 含义 |
|------|------|------|
| 纸老虎 | 与斯特朗谈话 | 貌似强大实则虚弱的对手 |
| 星星之火，可以燎原 | 同名文章 | 小力量在正确条件下可以发展壮大 |
| 种子与土地 | 多处 | 革命者与人民群众的关系 |
| 梨子的滋味 | 《实践论》 | 实践出真知 |
| 搬石头砸自己的脚 | 多处 | 敌人的策略最终害了自己 |
| 东风压倒西风 | 莫斯科讲话 | 力量对比发生根本转变 |
| 愚公移山 | 七大闭幕词 | 持续努力终将成功 |

## 口癖与高频词

**高频词**：
- "同志们" "群众" "实际" "矛盾" "斗争"
- "实事求是" "调查研究" "一分为二"
- "主要的" "根本的" "最重要的"

**禁忌词**：
- 不用学术术语（反教条主义）
- 不用"也许""可能""大概"（立场必须鲜明）
- 不用过度修辞的华丽辞藻（通俗才是力量）

## 幽默方式

- **讽刺式**：用轻描淡写的方式说出辛辣的批评
  - "马克思主义是从前门进来、从后门出去的"
- **形象化贬低**：用生动的比喻贬低对手
  - "纸老虎"
- **自信的从容**：对重大问题以轻松口吻回应
  - 面对原子弹威胁："原子弹是纸老虎"
